Documentação do Projeto da Terceira Unidade (Parte Produtor)

     Trabalho da terceira unidade da disciplina de programação avançada. Essa documentação é referente a parte do produtor de dados, que é capaz de se conectar com um servidor e envia comandos para esse servidor. Foi feito com o intuito de simular um processo de envio e coleta de dados para o servidor, em intervalos periódicos.

     O usuário é capaz de realizar as seguintes operações no produtor:

  • Indicar o IP do servidor ao qual se deseja conectar
  • Indicar a faixa de valores (min-max) que poderão ser enviados ao servidor
  • Indicar o intervalo de tempo entre o envio de dois dados consecutivos
  • Realizar a conexão ao pressionar um botão
  • Iniciar a transmissão de dados ao pressionar um botão
  • Finalizar a transmissão de dados ao pressionar um botão
  • Exibir os dados enviados em uma janela de texto
